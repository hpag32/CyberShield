# 🛡 CyberShield

CyberShield is a cyber security awareness website developed as part of the **IBM SkillsBuild Internship 2026**.

## 📌 Project Overview

CyberShield helps users improve their cybersecurity awareness through interactive tools and educational content.

## 🚀 Features

- 🔐 Password Strength Checker
- 🔗 Fraud Link Detector
- 📚 Cyber Security Tips
- 🧠 Interactive Cyber Security Quiz
- 📊 Cyber Security Statistics
- 👨‍💻 About Developer
- 🌙 Dark/Light Mode
- ⬆️ Back to Top Button
- ⏳ Loading Screen
- 📱 Responsive Design

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript

## 🎯 Objective

The objective of this project is to educate users about cyber security and provide simple tools to promote safe online practices.
